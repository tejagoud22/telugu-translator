# Telugu → English Translator (Gemini - Free)

## Get your FREE Gemini API key
1. Go to https://aistudio.google.com
2. Sign in with your Google account
3. Click "Get API Key" → "Create API key"
4. Copy the key

## Setup in 3 steps

### 1. Add your Gemini API key
Open `src/App.jsx` and replace line 5:
```js
const GEMINI_API_KEY = "YOUR_GEMINI_API_KEY_HERE";
```

### 2. Install dependencies
Open this folder in VS Code terminal:
```
npm install
```

### 3. Run the app
```
npm run dev
```
Then open http://localhost:3000 in Chrome.

## Free tier limits
- 15 requests per minute
- 1,500 requests per day
- Completely free, no card needed

## Notes
- Use Chrome for mic support (te-IN Telugu)
- Mic works locally — Chrome will ask permission once, click Allow
